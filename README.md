# 📝 Task Manager
> **Master your time, elevate your output.** A streamlined productivity hub designed to turn chaos into clarity. This platform empowers users to organize their workflows, track progress in real-time, and master their daily schedules with a clean, intuitive interface.

[🚀 View Live Demo](https://tskapps.netlify.app/) | [📂 View Repository](https://github.com/Harish919/task_manager)

---

## ✨ Key Features

* **Task Orchestration:** Create, Edit, and Delete tasks seamlessly.
* **Status Lifecycle:** Track tasks through `New`, `In Progress`, `Completed`, and `Canceled` states.
* **Smart Filtering:** Tabbed navigation to view All, Completed, or Pending tasks instantly.
* **Responsive Dashboard:** Built with a "Mobile-First" approach using Bootstrap 5.
* **Interactive Modals:** Clean pop-ups for adding and editing task details without leaving the page.
